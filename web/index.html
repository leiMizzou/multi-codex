Recommended flow

Capture each team or subscription once, then reuse it

  1. Create a slot named after the team, workspace, or subscription you want to preserve.
  2. Open ChatGPT or Settings, then manually switch to the target team or subscription in the browser.
  3. In normal mode, use the slot launch button to open codex login. In proxy mode, the same button opens codex with the saved provider settings.
  4. After login or proxy validation, refresh the dashboard. The slot remains isolated and reusable.
